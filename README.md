# RS-quicklist
Projeto Quicklist - Desenvolvido durante a formação Full-Stack da Rocketseat 🚀

## O que é a Quicklist ?

A Quicklist nada mais é do que uma simples lista de compras feitas com JS — um app web leve que permite adicionar itens, marcar como comprados e remover itens da lista usando HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar novos itens à lista de compras.
- Marcar itens como comprados com checkbox.
- Remover itens da lista e exibir aviso de exclusão.
- Exibir uma lista inicial com exemplos prontos.

## Como usar

1. Digite um item no campo "Adicionar um novo item".
2. Clique em "Adicionar item".
3. Marque o checkbox para riscar o item como comprado.
4. Clique no botão de lixeira para remover o item da lista.

## Tecnologias usadas

- HTML
- CSS
- JavaScript
